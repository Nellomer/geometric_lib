# **Geometric lib**
Репозиторий содержит в себе пять файлов написанных на python. Файлы circle.py, square.py, triangle.py и rectangle.py содержат в себе функции вычисления площади и периметра для окружности, квадрата, треугольника и прямоугольника соответственно. Файл tests.py содержит Unit тесты к другим файлам.

## **Circle.py**

### *def area* 
Принимает значение радиуса заданной окружности и возвращает её площадь.

### *def perimetr* 
Принимает значение радиуса заданной окружности и возвращает её периметр.

## **Square.py**

### *def area* 
Принимает значение стороны квадрата и возвращает его площадь.

### *def perimetr* 
Принимает значение стороны квадрата и возвращает его периметр.

## **Triangle.py**

### *def area* 
Принимает значение стороны треугольника и его высоту, возвращает его площадь.

### *def perimetr* 
Принимает значение сторон треугольника и возвращает его периметр.

## **Rectangle.py**

### *def area* 
Принимает значение сторон прямоугольника и возвращает его площадь.

### *def perimetr* 
Принимает значение сторон прямоугольника и возвращает его периметр.

## **Tests.py**
Содержит в себе Unit тесты к предыдущим файлам.

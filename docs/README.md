# **Geometric lib**
Репозиторий содержит в себе два файла написанных на python. Файл circle.py и square.py содержат в себе функции вычисления площади и периметра для окружности и квадрата соответственно. 

## **Circle.py**

### *def area* 
Принимает значение радиуса заданной окружности и возвращает её площадь.

### *def perimetr* 
Принимает значение радиуса заданной окружности и возвращает её периметр.

## **Square.py**

### *def area* 
Принимает значение стороны квадрата и возвращает его площадь.

### *def perimetr* 
Принимает значение стороны квадрата и возвращает его периметр.
